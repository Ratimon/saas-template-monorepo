## Welcome

This is a documentation starter template powered by **SvelteKit**, **Tailwind CSS v4**, and **MDSvex**. It provides everything you need to build beautiful, fast documentation sites.

## Features

- **Markdown-powered** — Write docs in Markdown with full MDSvex support
- **Syntax highlighting** — Code blocks with Shiki (light/dark themes)
- **Auto-generated navigation** — Sidebar built from your file structure
- **Table of Contents** — Extracted from headings with scroll spy
- **Dark mode** — Built-in light/dark theme toggle
- **Full-text search** — Command palette with Cmd+K
- **Responsive** — Mobile-friendly with collapsible sidebars
- **Prev/Next navigation** — Automatic links between pages
